# Design Tokens

Design tokens for the Sources system.

> Tokens are not yet formalized. Below are values currently used in prototypes that will be promoted to tokens.

## Color

| Name | Value | Usage |
|------|-------|-------|
| Source link | `#2563EB` | Inline citation text and chips |
| Source chip bg | `#EFF6FF` | Chip background |
| Source chip border | `#BFDBFE` | Chip border |
| Intuit blue | `#0077C5` | Brand accent in cards |
| Positive | `#0D8040` | Pro indicators |
| Negative | `#DC2626` | Con indicators |

## Spacing

| Name | Value | Usage |
|------|-------|-------|
| Card radius | `12px` | Source card corners |
| Card padding | `16px` | Internal card padding |
| Chip padding | `4px 8px` | Inline chip padding |

## Typography

| Name | Value | Usage |
|------|-------|-------|
| Source domain | `13px / 400` | Domain text in citations |
| Source company | `13px / 500` | Company name in citations |
| Chip label | `11px / 600` | Text inside chips |
| Superscript | `10px / 700` | Superscript citation numbers |
