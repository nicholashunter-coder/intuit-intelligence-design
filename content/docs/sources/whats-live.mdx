The following screenshots show source treatments as they appear in the live Intuit Intelligence experience today.

## Collapsed Sources

The default state after a response is generated. Sources are summarized as a count link (e.g., "3 sources") beneath the response actions.

![Collapsed sources view showing "3 sources" link beneath a chart response](/intuit-intelligence-design/screenshots/sources-collapsed.png)

Key details:
- Source count is displayed as a text link with a chevron
- Follow-up suggestion chips appear below the sources link
- The response area includes feedback actions (thumbs up/down, copy, download)

## Expanded Sources

When the customer taps the sources link, the full list expands, grouped by category.

![Expanded sources showing QB data and Web sources](/intuit-intelligence-design/screenshots/sources-expanded.png)

Key details:
- **"Your QuickBooks data"** section lists QB reports with icons and report names
- Expired links show a "Link Expired" badge with an info icon
- **"Web sources"** section lists external references with favicons and site names
- Each source is individually tappable

## Dashboard Context

In the QuickBooks dashboard, Intelligence opens as a docked side panel. Sources appear inline within the panel alongside the full dashboard experience.

![Full QuickBooks dashboard with Intelligence panel showing sources](/intuit-intelligence-design/screenshots/sources-dashboard.png)

Key details:
- Intelligence panel is docked to the right side of the dashboard
- Sources drawer is visible alongside the main QuickBooks navigation
- The "Ask anything" input remains accessible at the bottom of the panel
- Business Feed cards and action chips are visible in the main content area
