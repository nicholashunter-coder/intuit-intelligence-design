### What is the Sources project?

Sources defines how Intuit Intelligence experiences display, attribute, and link to external web content that AI surfaces in its responses. It covers everything from inline citation styling to expanded source cards.

### Why do some sources show a favicon and others don't?

We attempt to resolve favicons via the page itself first, then fall back to Google's favicon API (`google.com/s2/favicons`). Some sites block direct page fetches (403/404), which means we can't extract a native favicon. In those cases the Google API fallback usually works, but quality and freshness vary. See [Source Metadata](/docs/research/source-metadata) for the full resolution strategy.

### What's the difference between the inline treatment options?

We explored five levels of visual weight — from a simple superscript number up to a full chip with favicon and domain. The tradeoff is attribution visibility vs. reading flow disruption. See [Inline Source Treatments](/docs/design/inline-treatments) for the full comparison.

### How do I run the card interaction prototype?

Open `intuit-ai-cards-prototype.html` from the [repo assets folder](https://github.com/nicholashunter-coder/intuit-intelligence-design/tree/main/assets) in a browser. It's a self-contained HTML file — no build step needed.
